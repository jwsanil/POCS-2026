Build Instructions - EC-DATA-Container New Relic Demo
=================================================

Project Type
------------
This project is a Maven-based Spring Boot application packaged as an executable JAR.

Main Technologies Used
----------------------
- Spring Boot 3.2.3
- Java 17
- PostgreSQL JDBC Driver
- New Relic Java API
- Custom EC-DATA-Container library

Build Steps
-----------
1. Open terminal in project root directory.
2. Run Maven clean build:

   mvn clean install

3. After successful build, generated jar will be available in:

   target/new-relic-demo.jar

Run Steps
---------
Start application with New Relic Java agent:

   java -javaagent:newrelic/newrelic-agent-9.1.0.jar -jar target/new-relic-demo.jar

Additional Execution Notes
--------------------------
Detailed execution steps are provided in:

   /docs/notes

Dependency Note
---------------
This project uses Spring Boot starter web as primary application framework.
Additional database connectivity is provided through PostgreSQL JDBC dependency.
Monitoring integration is enabled through New Relic Java API dependency.

Important
---------
Ensure:
- newrelic folder exists beside project jar
- newrelic.yml contains valid license key
- Java 17 is configured before build
