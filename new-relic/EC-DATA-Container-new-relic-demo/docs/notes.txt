6. Notes

7. How to Download New Relic Agent
- Login to https://one.newrelic.com
- Navigate to Integrations & Agents -> Java
- Download latest Java agent zip
- Extract zip and place 'newrelic' folder near project jar
- Update newrelic.yml with license key

8. How to Create Keys
- Login to New Relic account
- Open Administration -> API Keys
- Create License Key for agent reporting
- Create User Key for API access
- Copy generated keys into secure location

9. Security Recommendation
- Rotate exposed keys if these are production credentials.
